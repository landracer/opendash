Data Point Categories

Engine / OBD2 Data Points

These data points represent engine parameters and OBD2 PIDs. They provide real-time information about engine performance and operation.

GPS / Navigation Data Points

These data points provide location and navigation information including position, speed, heading, and timing data.

IMU / Motion Data Points

These data points represent motion and orientation information from the 6-axis IMU sensor.

Battery / BMS Data Points

These data points provide battery management system information including voltage, current, and state of charge.

System Data Points

These data points provide system-level information including temperature, memory usage, and connectivity status.

Usage in UI

Each data point is assigned to a specific screen section in the UI. The system supports:

  • Configurable data views - Choose which data points appear in each screen section
  • Unit conversion - Automatic conversion between °C/°F, kPa/BAR/PSI, km/h/MPH, km/mi
  • Min/max tracking - Session high/low displayed per gauge page
  • Shift-light blink - Arc flashes red/blue when RPM exceeds configurable threshold
