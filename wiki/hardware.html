OpenDash - Hardware Overview

Comprehensive hardware specifications and component details

System Components

OpenDash is a modular racecar dashboard system with four ESP32-S3 display nodes communicating over a shared I2C bus.

ESP32-S3 Specifications

Core Hardware

  • Architecture: 32-bit RISC-V
  • Clock Speed: Up to 240 MHz
  • Flash: 16 MB external flash memory
  • PSRAM: 8 MB external PSRAM (Octal)
  • Connectivity: WiFi 802.11 b/g/n, Bluetooth 5.0
  • Peripherals: I2C, SPI, UART, ADC, DAC, PWM, GPIO

Display Controllers

Center Display (ST7262 RGB)

  • Controller: ST7262 RGB
  • Interface: Parallel RGB interface
  • Resolution: 800×480
  • Color Depth: 16-bit color (RGB565)
  • Touch: GT911 capacitive touch controller

Left/Right Gauges (ST7701S)

  • Controller: ST7701S LCD controller
  • Interface: 3-wire SPI + RGB parallel
  • Resolution: 480×480
  • Color Depth: 16-bit color (RGB565)
  • Touch: GT911 capacitive touch controller
  • IO Expander: TCA9554 I2C IO expander

GPS Unit (CO5300)

  • Controller: CO5300 AMOLED controller
  • Interface: SPI
  • Resolution: 466×466
  • Color Depth: 16-bit color (RGB565)
  • Touch: GT911 capacitive touch controller

Communication Interfaces

I2C Bus Configuration

  • Bus Speed: 400 kHz (Fast Mode)
  • Master: Center display
  • Slaves: Left gauge (0x10), Right gauge (0x11), GPS unit (0x12), BMS (0x20)
  • Ports:
    • Port 0 (Master): SDA=15, SCL=7 — internal bus for TCA9554 IO expander + GT911 touch
    • Port 1 (Slave): SDA=4, SCL=16 — external bus to Center unit

Pin Mappings

Each node uses a specific pin configuration:

// Center Display (ST7262 RGB) - I2C Port 0
// SDA = GPIO 19, SCL = GPIO 20 (I2C Master)
// Touch: GT911 on I2C Port 0
// SPI for display: MOSI=GPIO 18, MISO=GPIO 17, SCK=GPIO 16, CS=GPIO 15
// Left/Right Gauges (ST7701S) - I2C Port 1
// SDA = GPIO 4, SCL = GPIO 16 (I2C Slave)
// SPI for display: MOSI=GPIO 18, MISO=GPIO 17, SCK=GPIO 16, CS=GPIO 15
// IO Expander: TCA9554 on I2C Port 0
// Touch: GT911 on I2C Port 0
// GPS Unit (CO5300) - I2C Port 1
// SDA = GPIO 4, SCL = GPIO 16 (I2C Slave)
// SPI for display: MOSI=GPIO 18, MISO=GPIO 17, SCK=GPIO 16, CS=GPIO 15
// Touch: GT911 on I2C Port 0

Peripheral Components

GPS Module (LC76G)

  • Protocol: Multi-constellation (GPS, GLONASS, BeiDou, Galileo)
  • Accuracy: Centimeter-level positioning
  • Update Rate: 10 Hz
  • Features:
    • Position, velocity, time (PVT) data
    • Real-time delta vs. best lap
    • Sector-based predictions

IMU (QMI8658)

  • Type: 6-axis IMU (3-axis accelerometer + 3-axis gyroscope)
  • Accuracy:
    • Accelerometer: ±2g to ±16g range
    • Gyroscope: ±125°/s to ±2000°/s range
  • Features:
    • 3-axis g-force measurement
    • Orientation tracking
    • Parachute deployment system

Power and Connectivity

Power Requirements

  • Supply Voltage: 5V (USB-C or battery)
  • Current Draw:
    • Center Display: ~150mA
    • Left/Right Gauges: ~100mA each
    • GPS Unit: ~120mA
  • Power Management:
    • Built-in voltage regulation
    • Power saving modes
    • Overcurrent protection

Connectivity Options

  • WiFi: ESP32 built-in 802.11 b/g/n
  • Bluetooth: ESP32 built-in 5.0
  • USB: USB-C for programming and data transfer
  • CAN Bus: Center unit supports onboard CAN for ECU communication
  • I2C: Shared bus for inter-node communication
